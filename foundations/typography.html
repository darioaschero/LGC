Typography

Four type roles. Each toggles between default (Publico Text body / Sanomat display) and alt (National for both) via data-typeset="alt" on any ancestor. Font-size scales by 1.16 and line-height by its inverse, so the computed line in pixels stays invariant across typesets and vertical rhythm holds when switching.
