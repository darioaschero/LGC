GC Design System · v0.1
Overview

A multi-axis design system for a WordPress classic theme.

Tokens cascade primitives → accent → style → mode → component. Each layer adds one decision; no combinatorial explosion. Three axes drive every visual: mode (light/dark, decided by the end user), accent (per-post colour, decided by the editor), style (per-template, decided by the developer).

Status

  • M1 Foundations — neutral primitives, semantic surfaces, spacing, light/dark
  • M4 Typography — type roles + rich-text with drop letter and per-element axes
  • M2 Accent axis — three accents
  • M3 Style axis — default / solid / tint / gradient
  • M5 Component library
  • M6 Example WP templates
  • M7 Port to PHP / classic theme structure
