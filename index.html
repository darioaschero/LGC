GC Design System · v0.1
01 — Primitives

Neutral

The foundational gray scale. The only raw color values in the system — every semantic token resolves to one of these. Perceptually scaled in OKLCH so steps feel evenly spaced to the eye, not just to the math.

02 — Semantic

Surface & Foreground

Five tokens. Components reference these and only these. Mode resolution is automatic via light-dark() — the same token returns the correct value in light or dark without any duplicated CSS.
